Scroll Progressbar

Info

  • 페이지 스크롤 시 scrollHeight 값 백분율 환산
  • 백그라운드 꾸밈 요소로 인하여 웹접근성(A11Y) 속성 불필요
  • 상단 고정(bar) 및 좌측 하단 원형 고정(round) 2가지 타입 설계 (고정 위치의 경우 추후 CSS 커스텀 가능)

data-progressbar-options

  • 디폴트옵션 : {"type": "bar","isLabel": false}
  • type : bar / round 택 가능
    선택 타입에 따라 구조 상단 .scroll-progress 마크업에 클래스 동적 추가 (.scroll-progress--type-bar / .scroll-progress--type-round)
  • isLabel : 진행율(%) 노출
  • 본 옵션 전체 또는 일부를 누락하여도 상기 디폴트 옵션과 유저 옵션(우선 적용)이 병합되어 오류를 방어합니다.
