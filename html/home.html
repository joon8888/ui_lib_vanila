❌No Library💥 순수 바닐라JS로만 작업한 스크립트 모듈입니다. Github 소스보기

  • 100% 개인 작업입니다.
  • 기본 UI 컴포넌트를 모듈 단위로 분리하고, 스타일과 스크립트를 각각 구조화했습니다.
  • 본 사이트는 프레임워크 없이 hash(#)와 iframe 태그를 활용하여 SPA형태로 구현하였습니다.
  • 각 컴포넌트는 data-컴포넌트명-options 속성을 지니며, 그에 따라 유저 기능 커스텀이 가능합니다. (예시: 접근성 속성 사용 여부 boolean 값 전달)

SPA 스타일 page-loader (개인 구현 로직 및 코드)

Mark-up

          <div class="page-loader">
  <aside class="page-loader__nav">
    <nav class="nav">
      <ul class="nav__menu">
        <li>
          <a href="#hash명" class="nav__menu__btn nav__menu__btn--load">HOME</a>
          </li>
      </ul>
    </nav>
  </aside>
  <div class="page-loader__container">
    <iframe id="page-frame" src="html 경로" frameborder="0" width="100%" height="100%"></iframe>
  </div>
</div>

Script

          loadPage () {
    const hash = location.hash.replace('#', '') || 'home'; // default hash 세팅 
    const url = `html/${hash}.html`; // 동적 경로 생성
    const frame = document.getElementById('page-frame');
    if (frame) frame.src = url;
  
    document.title = hash === 'home' ? 'Script Module Libs' : `${hash} | Script Module Libs`; // 브라우저 title 속성 동적 변경
  
  ... 이하 버튼 클릭 이벤트 시 위 변수 동적 변경
}
